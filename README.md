# lly-cli

![Npm 版本](https://img.shields.io/badge/lly-cli_v1.1.0-green)

## 📚 简介

- 用于快速搭建前端项目的命令行工具

## 📕 模板

- ✅ 快速生成 [Vue3+TypeScript+Vite+ElementPlus+Pinia+Axios+Mock的模板](https://gitee.com/codelyty/cli-template)

## 🍭 安装

```bash
npm install lly-cli -g
```

## 🚩 使用

```bash
# 使用模板创建项目
lly create

# 执行项目文件夹名称并选择模板创建新项目
lly create [projeactName]

# 更新脚手架到最新版本
lly update

# 查看脚手架版本
lly -v
lly --version
```
